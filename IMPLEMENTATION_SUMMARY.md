# 电流环示教模式 - 完整实施方案

## 📋 项目概述

本实施为 Dummy V2 机械臂添加了**专业的电流环示教模式**，解决了之前"掰不动"的问题，让拖动示教变得轻松可控。

## ✅ 已完成的工作

### 1. 主控板固件修改 (STM32F4)

**修改文件：**
- `Core-STM32F4-fw/Robot/instances/dummy_robot.h`
- `Core-STM32F4-fw/Robot/instances/dummy_robot.cpp`
- `Core-STM32F4-fw/UserApp/protocols/ascii_protocol.cpp`

**新增功能：**
- `SetTeachMode(bool _enable, float _current)` 方法
- `!TEACH <current>` ASCII 命令（进入示教模式）
- `!TEACHOFF` ASCII 命令（退出示教模式）

**实现原理：**
```cpp
// 进入示教模式
void DummyRobot::SetTeachMode(bool _enable, float _current)
{
    if (_enable) {
        // 给所有关节发送 CAN 命令 0x03 (SetCurrentSetPoint)
        // 关节驱动板自动切换到电流模式
        for (int i = 1; i <= 6; i++)
            motorJ[i]->SetCurrentSetPoint(_current);
    } else {
        // 获取当前位置，切换回位置模式
        UpdateJointAngles();
        for (int i = 1; i <= 6; i++)
            motorJ[i]->SetAngle(currentJoints.a[i - 1]);
    }
}
```

## 🔍 与传统方法对比

### 之前的 `!DISABLE` 方法

```
问题：
- 直接关闭电机驱动器
- 可能有残余阻力
- 无法调节阻力大小
- 松手后位置不保持
```

### 新的 `!TEACH` 电流环方法

```
优势：
- 恒定的可预测阻力
- 电流值可调（0.1-2.0A）
- 有弹性，手感好
- 松手后基本保持位置
- 更安全（不会突然失控）
```

## ⚠️ 注意事项

### 固件兼容性

- **新固件**：支持 `!TEACH` 和 `!TEACHOFF` 命令
- **旧固件**：PC 端会自动降级到 `!DISABLE` 模式

### 安全提示

1. 进入示教模式前，确保周围无障碍物
2. 首次使用，建议从 0.5A 开始尝试
3. 拖动时双手稳住机械臂
4. 紧急情况下发送 `!STOP`

### 关节差异

- J1/J6 减速比 50:1，即使 0.5A 也有一定阻力
- J2/J3 受重力影响，注意下垂问题
- J4 减速比最小（24:1），最轻
