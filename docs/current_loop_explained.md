# 电流环示教模式详解

## 什么是电流环？

电流环（Current Loop）是一种电机控制模式，让电机输出**恒定的电流/力矩**，而不是恒定的位置或速度。

### 类比理解

```
位置模式（正常）：像钢筋一样硬，你掰不动
    电机："我要保持在这个角度，谁也不许动我！"

电流环模式（示教）：像弹簧一样有弹性，你可以拖动
    电机："我只用0.5A的力气，你能掰得动就掰吧"
```

## 电流环工作原理

### 电流-力矩关系

```
电流(A)    力矩感受
─────────────────────────────────
0.0A       零扭矩，完全自由（但有摩擦力）
0.2A       很轻，像转动阻尼很小的旋钮
0.5A       适中，像转动有一定阻力的旋钮  ← 推荐值
0.8A       较重，像转动较紧的旋钮
1.0A       很重，接近正常工作状态
2.0A       额定电流，正常工作
```

## 为什么之前掰不动？

### 问题分析

**之前使用 `!DISABLE` 命令：**

```
!DISABLE 的工作方式：
    发送 CAN 命令 0x01 (SetEnable false)
    关节驱动板：关闭电机驱动芯片
    结果：电机线圈断电，但可能有刹车或残余磁力
```

**问题：**
1. 某些关节驱动板版本在 `DISABLE` 后仍有保持电流
2. 机械结构的静摩擦力
3. 减速比大（J1: 50:1, J2/J3: 30:1），阻力被放大

**新的 `!TEACH` 命令：**

```
!TEACH 0.5 的工作方式：
    发送 CAN 命令 0x03 (SetCurrentSetPoint 0.5A)
    关节驱动板：
        1. 切换到 MODE_COMMAND_CURRENT
        2. 设置电流设定点为 0.5A
        3. 电机输出恒定的 0.5A 电流
    结果：电机有固定的"弹性"阻力，可以被克服
```

## 使用建议

### 不同场景的电流设置

```
场景1：快速示教（粗略动作）
─────────────────────────────────
电流：0.2 ~ 0.3 A
感受：非常轻，单手即可拖动
适用：大范围运动，快速示教
注意：松手后关节可能会因重力滑动

场景2：精确示教（精细动作）
─────────────────────────────────
电流：0.5 ~ 0.8 A  
感受：有适度阻力，可以精确控制
适用：精细操作，需要保持位置
优点：松手后基本保持位置

场景3：力控演示
─────────────────────────────────
电流：1.0 ~ 1.5 A
感受：较重，但仍在可拖动范围
适用：演示力控特性，测试外力响应
```

### 关节差异


## 进阶：电流环的物理原理

### 步进电机的力矩公式

```
T = Kt × I

其中：
  T = 输出力矩 (Nm)
  Kt = 电机力矩常数 (Nm/A)
  I = 相电流 (A)

对于 Dummy V2 的步进电机：
  额定电流 2A 时，额定力矩约 0.5 Nm
  0.5A 时，理论力矩约 0.125 Nm
  
考虑减速比后（以 J2 的 30:1 为例）：
  关节输出力矩 = 0.125 Nm × 30 = 3.75 Nm
  
这意味着你需要约 3.75 Nm 的力矩才能转动 J2
```

### 为什么电流越小越容易拖动？

```
电流减小 ──▶ 电机输出力矩减小 ──▶ 你需要克服的阻力减小

直观理解：
  0.5A：电机用"一根手指"的力气抵抗你
  0.2A：电机用"半根手指"的力气抵抗你
  0.0A：电机不用力，只有摩擦力和重力
```

## 总结

电流环示教模式的优势：

1. **可预测**：电流恒定，阻力恒定，手感一致
2. **可调**：通过调整电流值，获得不同的阻力感受
3. **安全**：即使碰撞，电机也只输出设定的电流，不会突然发力
4. **精确**：可以精确控制拖动的手感，适应不同场景

对比之前的 `!DISABLE` 模式：

| 特性 | !DISABLE | !TEACH (电流环) |
|------|----------|-----------------|
| 原理 | 关闭驱动 | 恒流输出 |
| 手感 | 不可控，可能有残余阻力 | 可控，弹性阻力 |
| 位置保持 | 无 | 有一定保持力 |
| 可调性 | 不可调 | 电流可调 |
| 适用性 | 简单场景 | 专业示教 |

---

**建议**：始终使用 `!TEACH` 命令进行示教操作，获得最佳体验！
