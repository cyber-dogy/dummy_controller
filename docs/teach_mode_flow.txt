================================================================================
                          关键 CAN 命令说明
================================================================================

┌──────────┬─────────────────────────────┬────────────────────────────────────┐
│ Cmd ID   │ 名称                        │ 说明                               │
├──────────┼─────────────────────────────┼────────────────────────────────────┤
│ 0x01     │ SetEnable                   │ 使能/禁用电机驱动器                 │
│ 0x03     │ SetCurrentSetPoint          │ 设置电流设定点 → 切换电流模式       │
│ 0x05     │ SetPositionSetPoint         │ 设置位置设定点 → 切换位置模式       │
│ 0x07     │ SetPositionWithVelocityLimit│ 带速度限制的位置控制                │
└──────────┴─────────────────────────────┴────────────────────────────────────┘

================================================================================
                          关节电流分配建议
================================================================================

【所有关节使用相同电流】

    电流值: 0.5A (推荐)
    
    效果:
        J1 (50:1) ──▶ 稍重，但可以拖动
        J2 (30:1) ──▶ 适中，注意重力
        J3 (30:1) ──▶ 适中，注意重力
        J4 (24:1) ──▶ 较轻
        J5 (30:1) ──▶ 适中
        J6 (50:1) ──▶ 稍重，但可以拖动

【如果固件支持独立控制】

    J1/J6: 0.3A  (减速比大，需要更小电流)
    J2/J3: 0.6A  (对抗重力，需要更大电流)
    J4/J5: 0.5A  (标准值)

================================================================================
                          安全注意事项
================================================================================

⚠️  进入示教模式前：
    ✓ 确保机械臂周围没有障碍物
    ✓ 确保有充足的空间拖动
    ✓ 建议先用小电流(0.3A)测试

⚠️  拖动过程中：
    ✓ 用双手稳住机械臂
    ✓ 避免突然松手
    ✓ 注意关节限位(J3最小35°)

⚠️  紧急停止：
    ✓ 发送 !STOP 命令
    ✓ 或点击 GUI 急停按钮
